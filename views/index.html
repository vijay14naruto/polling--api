Pooling System API

Features

1. Create a question
http://localhost:2000/questions/create
2. Add options to a question
http://localhost:2000/questions/id/options/create
3. Add a vote to an option of question
http://localhost:2000/options/:id/add_vote
4. Delete a question
http://localhost:2000/questions/id/delete
5. Delete an option
http://localhost:2000/options/id/delete
6. View a question with it’s options and all the votes
http://localhost:2000/allquestions
